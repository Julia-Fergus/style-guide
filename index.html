Buttons

A button is a call to action component.

Types

Primary Neutral Link Destructive

Primary buttons are used for main actions. The rest of the actions will be triggered by neutral buttons. In case an action is linked to a specific a link button will be displayed. Negative actions are represented by destructive buttons.

Height: 30px
Minimum Width: 100px (Actual width will vary depending on text)
Padding (Left & Right): 15px
Margin: 15px 5px

As more text is added to the button the width of the button increases, but all the numbers stay the same.
